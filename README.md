## Features

### v0.1

- [ ] Local Subscriptions
	No User-Login. Users can subscribe to channels locally, the data is stored on the device.
- [ ] Video Page
	- [ ] Video Playback
		- [ ] HTTP-Streaming
		- [ ] P2P Streaming (explore, optional)
		- [ ] Quality Selection
		- [ ] PIP
		- [ ] Native Controls and Player
	- [ ] View Title
	- [ ] View Description
	- [ ] View Comments (optional)
- [ ] Channel Page
	- [ ] View Channel Videos
- [ ] Subscription Page
	- [ ] View Subscribed Videos chronologically
